# Таблиця стандартних алгоритмів сортування в мовах програмування

| Мова програмування | Стандартна функція                 | Алгоритм за замовчуванням         | Стабільність | Типи даних                  | Особливості                                                                 |
|--------------------|------------------------------------|-----------------------------------|--------------|-----------------------------|------------------------------------------------------------------------------|
| Python             | `sorted()`, `.sort()`              | Timsort                           | ✅ Стабільна | Списки, ітеровані об'єкти   | Адаптивний, стабільний, ефективний на частково відсортованих даних         |
| Java               | `Arrays.sort()`, `Collections.sort()` | Dual-Pivot Quicksort / Timsort | ✅ / ❌       | Масиви, списки              | Timsort для об'єктів, Quicksort для примітивів                              |
| C++                | `std::sort()`                      | Introsort                         | ❌ Нестабільна | Масиви, вектори             | Гібрид Quick Sort + Heap Sort + Insertion Sort                              |
| JavaScript         | `Array.prototype.sort()`           | Quicksort / Timsort               | ❌ Нестабільна | Масиви                      | Реалізація залежить від рушія (V8 — Timsort для об'єктів)                   |
| C# (.NET)          | `Array.Sort()`, `List<T>.Sort()`   | Introsort                         | ❌ Нестабільна | Масиви, списки              | Висока продуктивність, нестабільна                                          |
| Go                 | `sort.Sort()`                      | Quicksort                         | ❌ Нестабільна | Слайси                      | Простий Quicksort без стабільності                                          |
| Rust               | `sort()`, `sort_unstable()`        | Timsort / Quicksort               | ✅ / ❌       | Вектори                     | Дає вибір між стабільною та швидкою нестабільною сортуванням               |
| Swift              | `sort()`, `sorted()`               | Introsort                         | ❌ Нестабільна | Масиви                      | Висока продуктивність, нестабільна                                          |
| Ruby               | `Array#sort`                       | Quicksort                         | ✅ Стабільна | Масиви                      | Зберігає порядок елементів з однаковими значеннями                          |
| PHP                | `sort()`, `asort()`                | Quicksort                         | ✅ / ❌       | Масиви                      | `asort()` — стабільна, `sort()` — ні                                        |
| Kotlin             | `sorted()`, `sort()`               | Timsort                           | ✅ Стабільна | Списки, масиви              | Наслідує поведінку Java                                                     |
| Scala              | `sorted`, `sortWith`               | Merge Sort / Timsort              | ✅ Стабільна | Колекції                    | Залежить від типу колекції та версії Scala                                  |
| Haskell            | `sort`, `sortBy`                   | Merge Sort                        | ✅ Стабільна | Списки                      | Функціональна реалізація, стабільна                                         |
| R                  | `sort()`, `order()`                | Shell Sort / Radix Sort           | ✅ / ❌       | Вектори                     | Для чисел — Radix, для загальних — Shell Sort                               |
